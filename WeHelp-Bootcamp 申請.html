兩百字內的個人簡介。

小弟今年27歲，目前待業中。
原本的工作是從事PVC鍍膜相關工作，但受近幾年疫情影響，公司出現一些問題，便決定轉換跑道。
由於本身會一些基本的電腦組裝，被老闆要求製作一個公司的官網，或許覺得都是電腦相關的我應該會懂，當時也沒多想，錢來就幹。
根據教學使用Wordpress建立和購買域名，主機等等...
算是第一次接觸到了PHP HTML CSS JS，整個過程覺得蠻有趣的，完成後也順利交差。
辭職後開始自學Web相關知識，決定朝著這個方向努力！

請描述一件你曾經碰到最困難的事情，如何克服？

應該是第一次用Wordpress架站的時候遇到的那些事。
當初主機是選擇GCP（Google Cloud Platform）VPS(單純因為有提供1年$300的測試額度，之後才知道有LocalHost這東東)
印象中那時候裝了某個優化的Plugin導致前後台都掛了進不去，然後開始找解決辦法，什麼用Cpanel找文件路徑把Plugin先停用，找了半天找不到。
後來終於搞懂我的VPS沒有Cpanel，研究了一下SSH和FTP，順利連到自己的主機上，停用了出問題的Plugin，網站才終於正常。
後續包括域名,DNS,CloudFlare,主機IP，透過網路上的資源慢慢摸索，網站上線後是選擇共享主機方便日後維護。
中間不斷的遇到問題，然後找方法，一直循環，整個過程對我來說不是一件容易的事。

關於這份申請網頁，請分享一個你開發時的技術心得。

一開始關於頁面佈局也是想了一陣子，後來是用Figma軟體先將我想要的頁面佈局畫出來。 按照畫的樣子，依序添加HTML和CSS，慢慢的讓網頁呈現我想要的樣子。
